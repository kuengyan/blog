Windows 11 倉頡選字問題救星：小狼毫 (Rime) 完整設定教學與回憶

老兵 十一月 25, 2025

如果你也是升級 Windows 11 後，發現內建的微軟倉頡輸入法候選字堆疊、無法順利選字，甚至懷念 Windows 7 時代的流暢感。這篇文章將分享如何透過開源工具「小狼毫 (Rime)」找回打字的手感。

🎞️ 我的打字啟蒙：從 386 電腦開始

我十七歲那年，一天晚上，老爸打電話告訴我，他朋友的女兒要來家裡借用電腦。那時家裡還是 386 電腦，螢幕是綠底黑字的老式顯示器。

她拿出一張 5.25 吋磁碟片，載入一份報告。我看著她熟練敲擊鍵盤，中文字一個個快速跳出螢幕。

好奇之下，我問她這速度要練多久？她說差不多一個學期。

基於不服輸，隔天我跑書局買了倉頡拆字規則書，一週勤練，打字速度竟超越她不少。

💔 靈魂的失去：Windows 11 的「選字惡夢」

我正式踏入了倉頡輸入法的世界。倉頡不僅成了我打字的工具，更融入了我的工作日常，成為身體記憶的一部分。

在 Windows XP、7 甚至 10 時代，倉頡依然保有那份智慧與流暢。它能根據上下文智能判斷詞彙，幾乎不需要手動選字。每當敲擊鍵盤，詞彙如流水般準確跳出。

然而，在 Windows 11 的某次版本更新後（尤其在 2025 年 7 月的 KB5062553 更新前後），一切都改變了。

內建倉頡彷彿失去了「靈魂」，退回到了最原始狀態——所有同碼字堆疊在一起，逼得我每次必須停下來，透過 數字鍵 逐一挑選。

那種暢快行雲流水的「頓」感被打斷，徹底粉碎了所有打字的韻律。

🛠️ 靈魂的找回：「小狼毫 (Rime)」輸入法調校教學

幾乎要放棄時，我找到了「小狼毫（Rime）」——一款開源且可自訂的輸入法引擎。我基於前人的基礎，改成更適合自己使用習慣的配置。

第一步：下載並安裝「小狼毫」主程式

您必須先有主程式。請至官方網站下載：

⬇️ 小狼毫 (Windows版) 官方下載頁

*依預設選項安裝，並選擇「倉頡五代」，完成後電腦會新增「小狼毫」輸入法。

第二步：站在巨人肩膀上 — 我的配置基礎

光有小狼毫還不夠。我那年代學的是「倉頡三代」，如今主流是「五代」，拆碼有差異。

我的目標：在小狼毫上，打造一個既兼容「倉三」習慣，又能擁有智慧選詞功能的輸入環境。

我參考了兩個 GitHub 專案：
  • rime-cangjie3-extension：讓五代也能用三代拆碼。
  • RimeCangjieInputConfiguration：優化詞庫與排序。

🚀 第三步（精華）：直接下載我的整合配置包

您無需自行研究，我將「倉三延伸碼表」與「台灣在地詞庫」整合打包好了。

📝 安裝步驟教學：

  1. 解壓縮檔案：
    下載並解壓縮 小狠毫輸入法倉頡Patch_20251111.7z。
  2. 開啟選單：
    在電腦右下角系統匣中的「小狼毫」圖示上按滑鼠右鍵。
  3. 複製 user 資料夾：
    • 點選選單中的「用戶文件夾」。
    • 將解壓縮包 user 目錄內的所有檔案，拷貝並覆蓋進去。
  4. 複製 data 資料夾：
    • 點選選單中的「程序文件夾」。
    • 將解壓縮包 data 目錄內的所有檔案，拷貝並覆蓋進去。
  5. 最後一步：重新部署
    再次在圖示按右鍵，點擊「重新部署 (Deploy)」。部署完成後即可享受流暢的打字體驗！

🏁 結語：黑白世界，回復色彩

經過這些設定調整，打字流暢感終於回來了。
